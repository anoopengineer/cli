GoCommons CLI Features

Let's see why GoCommon CLI is better than other CLI libraries…

Great subcommand support

Ever seen applications like git which supports multiple subcommands like git clone and git push? Well, now you can build you own myawesomeapp supporting multiple subcommands foo and bar effortlessly using this library

Full blown REPL terminal built in

Run your app without providing a subcommand and you get a builtin internal REPL terminal with full history, configurable prompt, keyboard shortcuts and autocompletion

Complete bash shortcuts supported in the REPL terminal

All the keyboard shortcuts that you are familiar with - up and down arrows, Ctrl-p, Ctrl-n to navigate between history, Ctrl-a, Ctrl-e and other text navigation shortcuts work out of the box

Supports colors in terminal

Build command line applications that supports displaying text in multiple colors, bold, italics, blinking with the colors package

Supports custom themes

Don't like the default colors? Create your own with Themes package
